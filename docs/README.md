# OBDDEVOPs QUANTUM LAB

Site web officiel du laboratoire quantique OBDDEVOPs.

## Structure du projet
```
/obddevops-quantum-lab
├── css/ # Styles CSS organisés par modules
├── js/ # JavaScript modulaire
├── assets/ # Images, polices, SVG
├── data/ # Fichiers de données JSON
└── docs/ # Documentation

```

## Installation

```bash
# Cloner le projet
git clone [url-du-projet]

# Installer les dépendances
npm install

# Démarrer le serveur de développement
npm run dev

# Build pour production
npm run build
